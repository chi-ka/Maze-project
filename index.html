THE MAZE - Raycasting Portfolio Project

Explore the fundamentals of 3D rendering with raycasting and SDL2.

Key Features

Walls

Create a window with SDL2 and use raycasting to draw walls on your window.

Orientation

Draw walls in different colors based on their orientation (NORTH, SOUTH, EAST, WEST).

Rotation

Provide a way to rotate the camera during execution using keyboard or mouse inputs.

Movement

Allow the player to move the camera using w, a, s, d keys and handle wall collisions.

About This Project

This project was inspired by the classic 3D rendering techniques used in early video games like Wolfenstein 3D. It is a portfolio project for alx fullstack engineering program, showcasing the basics of raycasting and SDL2 for rendering 3D environments.

The project is ment to higligth all the skills gained in this course so far.

The Game is set around a player who tries to get his way out of a maze built with texture brick, all the while avoiding obstacles that have been set in its path.
